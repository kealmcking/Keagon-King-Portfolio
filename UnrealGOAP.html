Game Info

UEGOAP is a small Goal-Oriented Action Planning (GOAP) system I built in Unreal Engine 5.7 using C++. It's a clean, minimal implementation focused on making the planning process easy to understand, debug, and extend.

The project separates planning from execution, uses an explicit world state to drive decisions, and supports multiple competing goals that the agent can react to as the world changes. When actions fail or conditions change, the agent replans automatically.

UEGOAP is meant as a focused AI systems project, not a full-fledged game, and exists to demonstrate solid GOAP fundamentals, clear architecture, and practical game AI decision-making.

Features Implemented

• Goal-Oriented Action Planning System

• Explicit World State Representation

• Priority-Based Goal Evaluation

• Reusable Action Definitions

• Planning vs Execution Seperation

• Autonomous AI Agent Ownership

• Failure Detection & Replanning

• Minimal Interactive World Actors

• GOAP Debug Visualization

• Deterministic, Explainable Architecture
