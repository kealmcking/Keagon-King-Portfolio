Game Info

Murder at Hollowstone Manor is a puzzle murder mystery game that was made as a portfolio project for my degree. The goal was to try and make a full game within a month, and while we may not have hit that mark, I think we made something very interesting.

While I helped with many aspects of the game, my main focus was the player controller, the camera, the dialogue system and the input system. The player base had been created by another classmate, and then I came in and expended it a lot, wrote a custom camera solution, as I was having an issue getting Cinemachine to work the way I wanted it to, and integrated all of the input controls into it.

The dialogue system was built from the ground up by me in order to allow us to speak and gain information from the suspects in order to reach our conclusion. As the month went on, the game design shifted and the dialogue system had less and less of a major role, however it still had a place in the finished product.
The input system was set up by me. I used the New Unity Input System as a base, and built out a system to allow for seamless switching of input types, and having that be reflected in the UI. Additionally, I built it in a way to handle enabling and disabling certain inputs when necessary, leaving that system to be handled by the input system itself instead of spread around the whole project.

Overall, it was a very enlightening experience where I learned a lot about working in a larger group under a tight deadline.
